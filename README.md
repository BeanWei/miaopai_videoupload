秒拍号视频上传工具  
=====
# 功能：
	1. 使用官方的上传接口，将本地视频上传至服务器，对官方中所有的选项都可以使用。
	2. 可对最近上传的视频进行删除，预览。  
	3. 自动选择与文件名同名图片 或 自动生成视频封面并上传。  
	4. 选中多文件可同时上传，或在队列中等待上传。  
	5. 对正在上传及完成的视频进行实时监控。
